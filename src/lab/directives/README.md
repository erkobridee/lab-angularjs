# Angular.js Directives

> Diretivas é o que possibilita extender o HTML e criar tags  customizadas, possibilitando criar componentes, os quais poderão ser  reaproveitados em futuros projetos.

* [Creating Custom Directives | Angular.js Guide](https://docs.angularjs.org/guide/directive)


## Examples

* [Angular.js](angularjs)

  * [combobox](angularjs/combobox)

  * [datagrid](angularjs/datagrid)

* [Chart.js](chart.js)  

* [JQuery UI](jquery-ui)

* [Polymer](polymer)  

* [React.js](reactjs)
